# Set up CocoaPods

## Checkout branch

    $ git checkout -b cpod

## Links
* https://guides.cocoapods.org/using/getting-started.html
* https://guides.cocoapods.org/using/using-cocoapods.html

## Install CocoaPods
    $ sudo gem install cocoapods

## Setup Project
    $ pod init
    $ pod install
